# Unity WebGL build

Ce dossier doit contenir l'export Unity WebGL.

Attendu apres export:

- Build/game.loader.js
- Build/game.data
- Build/game.framework.js
- Build/game.wasm

Export Unity:

1. Ouvrir le projet unity-project dans Unity.
2. Menu File > Build Settings > WebGL > Build.
3. Copier le dossier exporte dans ce dossier public/unity.
